## Technology Stack

-   Laravel
-   Jetstream + Inertiajs
-   Vue3
-   MySQL

## Setup locally

-   git clone https://github.com/joppx25/product-order-inertia.git

```bash
  composer install
  npm install
  php artisan migrate --seed
```

## Access locally

```bash
 - http://localhost:8000
 - Register account in http://localhost:8000/register
```
